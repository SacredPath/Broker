CSS Loading Test

If you see styled content, CSS is loading correctly!

Check browser DevTools (F12) -> Network tab to see if CSS files are loading.

CSS Files Should Load:

  • bootstrap.min.css
  • style.css
  • aos.css
  • all.min.css
  • swiper-bundle.min.css

Troubleshooting:

  1. Open browser DevTools (F12)
  2. Go to Network tab
  3. Refresh the page
  4. Look for CSS files - check if they load (status 200)
  5. If 404 errors, the server might not be running from the correct directory
